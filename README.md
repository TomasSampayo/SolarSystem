# SolarSystem 
SolarSystem es una app con información concisa y fácil de leer sobre los planetas y planets enanos de nuestro sistema solar. 

## Project Setup

### Instalar Node
Instalar Nodejs (https://nodejs.org).

### Para instalar las dependencias del proyecto
```npm install ```

### Para compilar
```npm run serve```
